# Anki Plus

Минималистичное MVP-приложение для изучения французской лексики с короткими уроками, упражнениями в стиле современных vocabulary drills и локальным сохранением прогресса.

## Стек

- React
- TypeScript
- Vite

## Запуск

Требуется установленный `Node.js 20+`.

```bash
npm install
npm run dev
```

Прод-сборка:

```bash
npm run build
npm run preview
```

## Что внутри

- Главный экран с прогрессом дня, серией, новыми словами и быстрыми разделами
- Урок на 20 заданий с 4 типами упражнений по схеме французский -> русский
- Автовоспроизведение французского слова через browser TTS и кнопка повтора
- Мгновенная проверка ответа и показ правильного варианта
- Экран результата с ошибками и режимом `Повторить ошибки`
- Сохранение прогресса слов и дневной статистики в `localStorage`
