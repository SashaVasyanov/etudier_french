# Etudier French

Минималистичное приложение для изучения французской лексики с короткими ежедневными уроками, модульным прохождением и локальным сохранением прогресса. Теперь проект можно запускать и как desktop-приложение через Tauri.

## Стек

- React
- TypeScript
- Vite
- Tauri

## Требования

- `Node.js 20+`
- `npm 10+`
- `Rust` и `Cargo` через `rustup`
- Для macOS: `Xcode Command Line Tools`

```bash
node -v
npm -v
rustc --version
cargo --version
```

Если Rust не установлен:

```bash
curl https://sh.rustup.rs -sSf | sh
source "$HOME/.cargo/env"
```

Для macOS при необходимости:

```bash
xcode-select --install
```

## Установка зависимостей

```bash
npm install
```

## Запуск в браузере

```bash
npm run dev
```

## Запуск desktop-версии через Tauri

```bash
npm run tauri:dev
```

Tauri поднимает существующий Vite dev server на `http://127.0.0.1:1420`, а затем открывает приложение в отдельном desktop-окне. Данные по-прежнему сохраняются локально через `localStorage`, но уже внутри WebView desktop-приложения.

## Сборка web-версии

```bash
npm run build
npm run preview
```

## Сборка desktop-приложения

```bash
npm run tauri:build
```

## Что внутри

- Главный экран с понятным следующим действием: ежедневный урок, словарь, прогресс
- Ежедневный урок, разбитый на отдельные модули с прогрессом по текущему дню
- Словарь со статусами слов: новые, изучаемые, известные, выученные, сложные
- Возможность пометить слово как `Уже знаю`, чтобы оно не попадало в новые слова
- Сохранение прогресса, статистики и состояний слов в `localStorage`

## Как работает интеграция Tauri

- UI остаётся на React + TypeScript + Vite.
- В режиме `tauri:dev` Tauri использует существующий Vite dev server как источник интерфейса.
- В режиме `tauri:build` Tauri сначала запускает `npm run build`, затем упаковывает содержимое `dist/` в нативную desktop-оболочку из `src-tauri`.
- Логика уроков, словаря, прогресса и локального хранения не меняется и продолжает работать поверх текущего фронтенда.
